## 🧬 &nbsp;Кто я

Я — разработчик, которого зацепило сразу всё:
боты и автоматизация, железо и электроника, AI и нейросети.
Не выбирал одно направление — взял все и соединил.

Пишу инструменты, которые реально использую сам.
Управляю сервером через Telegram. Общаюсь с AI без интернета.
Слушаю радиоэфир через ESP32. Изучаю протоколы с Flipper Zero.

Каждый проект начинался с одной мысли:
*«Почему этого ещё не существует?»* — и заканчивался рабочим кодом.

```js
актботарт = {
    "зацепило":   ["боты", "железо", "AI", "автоматизация"],
    "инструменты": ["ESP32 + CC1101", "Flipper Zero", "локальные модели"],
    "проекты":    ["ACT Хостовик", "Линг", "ACT ВК"],
    "принцип":    "строю то, чем сам хочу пользоваться",
}
```

### 🖥️ &nbsp;[ACT Хостовик](https://t.me/act_man_bot) &nbsp;—&nbsp; сервер в кармане

</div>

Telegram-бот для управления сервером — без лишних приложений.
Полноценный SSH, мониторинг, PM2, бэкапы и алерты — всё прямо в чате.
Работает с любым хостингом.

```
💻 SSH терминал    →  полноценная работа с сервером прямо в Telegram
📊 Статистика      →  CPU, RAM, диск, сеть — одним взглядом
⚙️ PM2             →  старт · стоп · рестарт · удаление · логи · автозапуск
🔄 Обновление      →  апдейт системы в одну кнопку
🔧 Службы          →  логи, соединения, статус без лишних действий
🌐 Мониторинг      →  HTTP-проверка сайтов и сервисов
💾 Бэкапы          →  резервные копии по расписанию
🔔 Алерты          →  уведомление при падении и восстановлении сервера
```

### 🤖 &nbsp;[Линг](https://t.me/act_chat_bot) &nbsp;—&nbsp; AI-собеседник в Telegram

</div>

Бот, с которым можно говорить. Не «введите команду» — а просто написать и получить ответ.
Отвечает на вопросы, поддерживает разговор, помогает разобраться в чём угодно.
Работает на языковых моделях — быстро, без лишних шагов.

```
💬 Живой диалог    →  не команды, а обычное общение
🧠 AI-ответы       →  языковые модели под капотом
⚡ Без задержек    →  быстрый отклик на любое сообщение
🔒 Приватно        →  без сторонних облаков
```

## 🔧 &nbsp;Железо

Код — это только половина. Вторая половина — устройства, с которыми этот код работает.

```
📡  ESP32 + CC1101   —   Sub-GHz в руках. Слушаю эфир, разбираю протоколы,
                         строю собственные RF-решения с нуля.

🐬  Flipper Zero     —   Лучший способ понять, как устроен беспроводной мир.
                         RFID, NFC, IR, Sub-GHz — всё в одном корпусе.

🤖  Локальный AI     —   Языковые модели без интернета и без чужих серверов.
                         Если модель запускается у тебя — она точно твоя.
```
